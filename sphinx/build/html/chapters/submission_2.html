Tasks regarding the one-dimensional implementation of wave propagation

The task was to include the f-wave solver in the given files implementing wave propagation, including the test file. Additionally we should find a solution to switch between Roe and f-wave solver. The switch was implemented with MODE macros and switch functions in the respective files. The modes to choose from are MODEFWAVE and MODEROE, which then follow the chosen solver computations and tests.
